	Версия 5.7.2.275   06.03.2015
	-------------------------------------- 		
	- На сайте обновлен шаблон для импорта студентов из Excel файла (contingent.tdmu.edu.ua - Контингент - Скачать, или прямая ссылка: http://contingent.tdmu.edu.ua/res/6291/import.zip). 
	  Добавлены поля: 
	  "EDBO ID Person (Целочисленный идентификатор персоны в EDBO)" - необязательное поле; 
	  "Десятизначный код КОАТУУ" - адрес регистрации студента; 
	  "Город" - если код КОАТУУ указан, а это поле пустое, то оно может быть заполнено автоматически, если в коде КОАТУУ (который соответствует адресу), зашифрован хоть один населенный пункт типа: село, місто, селище, селище міського типу. Если зашифровано более одного, то возьмется последний населенный пункт в адресе из перечисленных типов; 
	  "Улица"; 
	  "Дом"; 
	  "Корпус"; 
	  "Квартира". 
	  Все поля - необязательные. На данный момент эти поля используются в Education для заявки документа: "студенческий билет - банковская карта". Поля включены в пакет, формируемый для программы Education. ВНИМАНИЕ! Перед импортом студентов с КОАТУУ кодами следует обновить справочник КОАТУУ кодов. Возможность обновления добавлена в справочниках, как импорт из *.DBF файла справочника, доступного на официальном сайте http://www.ukrstat.gov.ua/klasf/klasif/koatuu.rar. ВНИМАНИЕ! Перед импортом справочника обязательно выполнить резервное копирование базы Контингента.
	  Импорт из Excel начиная с этого обновления работает только с Firebird 2.5.

	- Доработан отчет "ReportsRu\Контингент по странам". Выводит контингент по выбранным подразделениям на дату. (требует Firebird 2.5)
	- Доработан отчет "Reports\Контингент по странам".  Выводит контингент по выбранным подразделениям на дату.  (требует Firebird 2.5)
