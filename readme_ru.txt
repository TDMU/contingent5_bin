*  Инсталляция Контингент 5  *
==============================

====================================================
Конвертирование базы Контингент 4.3.0.70 - 4.5.0.80 
в базу Контингент 5 (клиент-серверная версия)
====================================================
 
 !!!  Обязательно установить сервер баз данных Firebird v.1.5.6.5026   !!!
 !!!  http://firebirdsql.org/index.php?op=files&id=engine_156 !!! 
  
  1. Сделать резервную копию БД "Контингент 4".
  2. Восстановить базу данных Контингент 5 (contingent5.fdb) из транспортного формата (contingent5.fbk),(Пример полной команды: 
  "c:\data\gbak -R -USER SYSDBA -PASSWORD masterkey contingent5.fbk contingent5.fdb").
  3. С инсталляционного диска "Контингент 4" запустить программу "convert_v3-v4\IBPump.exe".
  4. Для загрузки профиля нажать "LoadProfile".
  5. В диалоге выбрать "cont4-5.ibp". Нажать "Открыть". 
  6. На закладке "Step1: DataBases" в поле "Source DB" выберите полный путь к базе "Контингент 4" "contingent4.gdb" в формате "сервер:путь к базе", в поле "User Name" должно быть "SYSDBA", в поле "Password" ввести пароль доступа к серверу баз данных. 
  7. На закладке "Step1: DataBases" в поле "Destination DB" выберите полный путь к восстановленной базе "Контингент 5" "contingent5.fdb" в формате "сервер:путь к базе", в поле "User Name" должно быть "SYSDBA", в поле "Password" ввести пароль доступа к серверу баз данных.
  8. На закладке "Step3: Pump" нажать кнопку "Start". В основном окне отображается протокол конвертирования базы. Этот процесс может занять достаточно длительное время. 
  9. Рекомендация. После конвертирования базы создать резервную копию базы данных "Контингент 5" "contingent5.fdb" и выполнить ее восстановление. 
  10. Деинсталлировать все клиент-серверные рабочие места "Контингент 4"
  11. Проинсталлировать все клиент-серверные рабочие места "Контингент 5" 

===============================================================================
Конвертирование базы Контингент 4.3.0.70 - 4.5.0.80 в базу Контингент 5 (персональная версия)
===============================================================================
Перед инсталляцией персональной версии "Контингент 5" нет необходимости удалять персональную версию "Конитнгент 4". 
  1. Следует внимательно проверить версию "Контингента 4" (4.3.0.70 - 4.5.0.80) При несоответствии следует установить обновления...
  2. При инсталляции "Контингент 5" будет предложено переконвертировать базу автоматически.
  3. Проверить корректность данных в Контингент 5.
  4. Выполнить деинсталляцию Контингент 4.
